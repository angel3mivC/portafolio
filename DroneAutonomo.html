Drone Autónomo

Desarrollé un algoritmo para que un dron autónomo siga una línea y tome decisiones basadas en la detección de formas utilizando OpenCV. El algoritmo procesa datos visuales para reconocer formas, lo que permite al dron ajustar su trayectoria o realizar acciones específicas, logrando una completa automatización sin control manual.

Tecnologías utilizadas

  • Python
  • OpenCV
  • HTML
  • CSS
