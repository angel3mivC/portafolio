Drone Autónomo

Desarrollé un algoritmo avanzado para que un dron autónomo siga una línea y tome decisiones en tiempo real utilizando detección de formas con OpenCV. Este algoritmo analiza datos visuales y reconoce formas específicas, permitiendo que el dron ajuste su trayectoria o ejecute acciones determinadas según el entorno. La implementación de esta solución logra una completa automatización del dron, sin necesidad de intervención manual, y demuestra el potencial de la visión por computadora para crear sistemas autónomos más inteligentes y eficientes.

Tecnologías utilizadas

  • Python
  • OpenCV
  • HTML
  • CSS
