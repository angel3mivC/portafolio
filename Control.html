Control de videojuegos

Desarrollé un control para videojuegos usando Arduino y Python, combinando hardware y software para crear una experiencia interactiva única. El proyecto emplea un microcontrolador Arduino que registra las entradas del usuario a través de botones y joysticks, y las envía a una aplicación en Python. La aplicación, a su vez, procesa estas señales para simular los comandos del teclado o el ratón, permitiendo la compatibilidad con distintos videojuegos. Este proyecto me permitió profundizar en la programación de hardware y explorar la integración de dispositivos físicos con aplicaciones de software, abriendo posibilidades para personalizar controles de juego de acuerdo con las necesidades del usuario.

Tecnologías utilizadas

  • Python
  • Arduino
